# Example: Deep Timeline Mechanism Extraction

Seed event: [[Bronze Age Trade And Collapse]]

Candidate mechanism: Interdependence amplifies shocks.

Useful because: systems optimized for high connectivity and elite throughput may become fragile when multiple inputs fail at once.

Falsifier: a network remains robust despite losing several hubs because it has redundancy, substitution paths, and local production capacity.

Non-application boundary: do not use “Bronze Age collapse” as a generic analogy for every modern supply-chain problem. Name the actual dependency graph, buffers, and failure channels first.

Promotion decision: keep as candidate until source spine is upgraded.
