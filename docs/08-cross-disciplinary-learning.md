# Cross-Disciplinary Learning

Object types:

- Concept: reusable named idea.
- Mechanism: causal loop.
- Motif: recurring symbolic/story pattern.
- Bridge note: useful connection between domains.
- Question: durable inquiry.
- Disagreement: unresolved conflict.
- Non-connection: tempting but false link.
- Feedback record: reality test.
- Counterfactual: what if the claim is false?
- Reasoning trace: why a conclusion was reached.

Resemblance is not proof. Cross-domain extraction creates candidates that must survive source grounding, critique, use, scoring, or clear application.
