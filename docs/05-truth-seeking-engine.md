# Truth-Seeking Engine

The best artifact is not more notes. It is verifiable proof of improving judgment.

Core surfaces:

- Prediction ledger: claim, domain, probability, invalidation, resolution source, result.
- Belief revision log: prior belief, confidence, evidence that changed it, new belief.
- Canonical claim gate: source spine, confidence, strongest objection, falsifier.
- Calibration retrospective: what systematic bias appeared and what correction rule follows.

Rule: a wrong call recorded honestly outranks a right call rationalized after the fact.
