Use clippings, source notes, synthesis reports, and a small skill patch ledger.

## Research harvester
Add source-quality ladder, promotion candidates, semantic review, and source queues.

## Agent-skill improver

Add authority maps, task/result packets, report routing, and Git-backed verification.

## Scheduled cockpit
Add loop scoreboard, script-first cron jobs, compact reports, and kill/pause triggers.

## Sandbox lab
Add a separate lab vault, run ledgers, decay reports, promotion packets, and objective scores.
