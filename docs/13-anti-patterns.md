# Anti-Patterns

- Vault as dumping ground.
- Skill files as diaries.
- Every clipping becomes a skill.
- Every interesting idea becomes memory.
- Chat/messaging as source of truth.
- More context = better reasoning.
- More automation = more intelligence.
- Agent edits its own evaluator.
- Pretty synthesis without changed behavior.
- No kill lane.
- No rejected-edit ledger.
- Canon without strongest objection.
- Token savings by globally weakening everything.
- Sidecars treated as truth.
- Sandbox outputs silently promoted.
- “Verified” used as prose instead of test result.
