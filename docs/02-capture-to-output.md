# Capture-to-Output Protocol

Capture is not progress. Capture preserves a possibility. Output resolves it.

Disposition labels:

- `use-now`: immediately changes a current artifact or decision.
- `route-project`: belongs to a named project packet.
- `skill-runbook`: should patch a reusable procedure.
- `experiment`: needs a bounded test.
- `source-to-read`: high-value source but not yet processed.
- `research-lead`: plausible lead requiring stronger evidence.
- `defer`: useful later, no current action.
- `kill/archive`: not worth more attention.
- `already-absorbed`: existing skill/runbook already covers it.

Clipping metabolism:

```text
raw clipping -> metadata/provenance -> claims -> reusable idea -> candidate patch -> validation gate -> accepted/deferred/rejected/killed
```
