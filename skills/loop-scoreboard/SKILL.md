# Loop Scoreboard


Use for cron jobs, reports, scouts, dashboards, clippings sweeps, monitors, and agent loops.

For each loop, record: owner, cadence, source of truth, last review, decision impact, signal quality, review burden, false-positive risk, time saved, verdict, next action.

Weekly question: What did this loop cause us to do differently?
